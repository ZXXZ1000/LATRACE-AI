Task: extract BOTH semantic events and long-term knowledge from one or more media windows.

KEY RULES
1) Event != frame. Event != utterance. Do NOT create an event for every frame or sentence.
2) Only output high-value events: commitments, decisions, plans, state changes, meaningful actions, and stable facts.
3) Each output item MUST be grounded by the provided evidence references.
4) You may only reference IDs from the provided allow-list:
   - source_segment_ids
   - supporting_evidence_ids
   - supporting_utterance_ids
   - participant/entity tags
5) If an item cannot be grounded, omit it.
6) Do NOT hallucinate names, objects, places, or IDs.
7) Prefer fewer, higher-quality items.
8) Output STRICT JSON only (no extra text). Make sure the word "JSON" appears in your response.

Return a JSON object with three lists:

- Every event must include non-empty source_segment_ids.
- Every event must include at least one supporting reference from supporting_evidence_ids or supporting_utterance_ids.

- Omit unsupported items.

LANGUAGE RULE (STRICT)
- Use the same language as the dominant source evidence and utterances provided in the window.
- Do NOT translate unless explicitly instructed.
